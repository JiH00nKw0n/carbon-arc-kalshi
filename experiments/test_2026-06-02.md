## 1. Kill-test on CA0034 (`s_h_leadlag_stats.py`)

CA0034 POS Volume(보유) × NFP, Core CPI(로컬 FRED 캐시). 5개 게이트:

| 테스트 | NFP | Core CPI | 의미 |
|---|---|---|---|
| 1. 정상성 ADF/KPSS | 비정상 (ADF p=0.84) | 비정상 (KPSS p=0.035) | YoY가 stationary 아님 → 상관 spurious 위험 |
| 2. Raw CCF (lag −6…+6) | **전 구간 r 0.78–0.82 평평** | 0.68–0.81 평평 | peak 없음. "+1 best"는 ±2 max 착시. 0/1/2 spread 0.009 |
| 3. Prewhitened CCF (+1) | **r=0.09** (band 0.29) | r=0.18 | 자기상관 제거하면 lead 소멸 |
| 4. Granger CA→macro | p=0.41/0.66/0.53 | p=0.95/0.29/0.30 | CA가 macro를 Granger-cause 안 함 |
| 4b. Partial Granger (Z 통제) | p=0.38 | p=0.027 | NFP 탈락, Core CPI 단발 통과 |
| 5. OOS Diebold-Mariano | p=0.72 | p=0.13 | AR(1) 대비 OOS 예측 개선 0 |
| + Pyper-Peterman 실효 N | N=45→**N_eff=5.8**, p 0.00→0.059 | N=44→**N_eff=5.6**, p 0.00→0.091 | "n=46"은 허수, 실효 ~6 |

VERDICT: 둘 다 lead 미생존.

## 3. 정직한 재분석 — AND-게이트 overclaim 교정 (`s_j_leadlag_robust.py`)

사용자가 "한 테스트 실패=가짜는 overclaim, 테스트마다 의미 다름"을 지적 → 타당. 존재(prewhitened/Granger) vs
유용성(OOS)을 뭉갠 것. 자의적 AND 대신 두 가지로 판별:

- **(A) BH-FDR** (partial-Granger p 15개 family): **0/15 survive.** 최소값 0.008도 기준 0.0033 미달.
- **(B) Circular-shift surrogate** (CA 자기상관 보존, 정렬만 파괴):
  - Granger-F emp_p: 전부 ≥ 0.077 (CA0034×core 0.077, CA0056×core 0.089, CA0056×pce 0.13 …) → **존재 미입증**
  - OOS emp_p: CA0056×pce=0.022, CA0030×retail=0.022 (<0.05)지만 **효과크기 ~1e-6** + 가장 큰 OOS 개선
    (CA0034×umich 1.58e-3)은 오히려 0.15 → 유의성이 효과크기 안 따라감 = noise 전형. 하나는 known-artifact(CA0030).

## 4. 결론

- **monthly lead는 "가짜"가 아니라 "demonstrable하지 않음 (noise 쪽으로 기움)".** FDR·Granger-surrogate 미통과,
  OOS의 2개 flicker는 상호 불일치·극소효과. 그러나 power 부족으로 "진짜 없음"을 증명하지도 못함.
- **근본 원인: 단일 매크로 사이클(2021–2026) → 독립 변곡점 ~1개 → N_eff≈6.** 같은 윈도우에서 데이터셋을
  더 사도(예산 $10,525) 결론 불변. 비선형으로 가면 power 더 악화(데이터 더 필요).
- **유일한 탈출구**: 독립 표본(이벤트) 수를 늘리는 것 = Kalshi event study. 단 "macro weekly"(타깃이 월간이라 무의미)가
  아니라 **"CA가 시장가가 못 담은 정보를 갖는가"를 발표 이벤트 다수에 걸쳐** 보는 것. → 후속 계획
  `~/.claude/plans/twinkly-roaming-popcorn.md` (Test α outcome study).

## 5. 방법론 메모

- 모든 테스트가 **선형**(Pearson/CCF/Granger/OLS/AR). 비선형 lead는 원리상 못 잡음 — 단 n=46/N_eff≈6에선
  비선형 검정이 power가 더 필요해 이 데이터엔 부적합. 비선형은 표본 늘린 뒤에야 의미.
- sign convention: r(k)=corr(CA.shift(k), macro), k>0 = CA leads.

## 7. 세팅 업그레이드(v2) + 신규 데이터 구매 — 같은 날 마지막

세팅 점검 요청 → α 타깃을 **`실현치 − 시장 내재기대값`(% 단위, strike 사다리 적분 → 필터링 불변)** + 피처를 **계약 horizon 맞춤(yoy/mom/dyoy)** 으로 업그레이드 (`s_o_alpha_v2.py`, `analysis_kalshi_alpha_v2.md`).

- **핵심: card→CPI candidate 가 artifact 였음이 드러남.** 구 타깃 mean(y−p)에서 r=+0.27 → **신 타깃에서 r=−0.01~−0.21 (부호까지 뒤집힘).** noisy 타깃 구성의 허상이었음. (β-WoW null 과 일관.)
- 신규 구매 (clean setup 으로 검증): **CA0060 Foot Traffic $29.69**(insight 45862 방문수) + **CA0028 Detailed Card $54.72**(insight 347 Inflowing spend).
- **CA0060 → NFP** (실물 고용 채널, 새 메커니즘 가설): **null** (r≈0, p>0.36).
- CA0060 → CPI/PCE: 음수 r (방문↑→인플레↓, 경제적 역방향 = spurious), FDR 탈락.
- CA0028 → CPI/PCE/NFP: 전부 null.
- **종합: 5 데이터셋 × 4–5 매크로 × 3 변환 = 57셀, FDR 생존 0개.** clean 방법론에서 전면 null.

**최종(갱신)**: 월간·α·α-v2·β-lead·β-WoW, 그리고 5개 데이터셋(POS·card×2·clickstream·foot traffic) 전부에서 **CarbonArc가 Kalshi 매크로 시장 대비 edge 없음.** 가장 그럴듯했던 두 후보(card→CPI, foot→NFP) 모두 각각 artifact / null 로 판명. 데이터·표본·방법 다 갖춘 negative.
